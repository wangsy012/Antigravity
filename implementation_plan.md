# 開發純 HTML 俄羅斯方塊 (Tetris)

本計畫將開發一個完全獨立、只需瀏覽器即可執行的純 HTML 俄羅斯方塊遊戲。採用單一 `index.html` 檔案包含所有結構、樣式及邏輯，並採用深色現代化、帶有些微玻璃擬態 (Glassmorphism) 的設計，以提供極佳的視覺體驗。

## User Review Required

> [!IMPORTANT]
> **音效實作方式**：為了達成「純 HTML」而不需要依賴外部 `.mp3` 或 `.wav` 檔案（避免跨域或檔案管理問題），本計畫將使用瀏覽器內建的 **Web Audio API** 即時合成音效（如移動、旋轉、硬降及消除音效）。請確認這是否符合您的期望，或者您會提供具體的聲音檔案素材？
> 
> **控制配置**：
> - `左 / 右箭頭`：左右移動
> - `下箭頭`：軟降 (加速下落)
> - `上箭頭`：順時針旋轉 (右轉)
> - `Z 鍵`：逆時針旋轉 (左轉)
> - `空白鍵 (Space)`：直接下落 (硬降 / Hard Drop)
> - 請確認以上的按鍵配置是否順手。

## Proposed Changes

### 核心檔案

#### [NEW] index.html (建立於工作區 `c:\Users\dante\OneDrive\桌面\tetrio`)
將建立單一 HTML 檔案，包含以下三個主要部分：

1. **HTML 結構**：
   - 包含主遊戲畫布 (`<canvas>`)，尺寸通常為 10 寬 x 20 高個方塊。
   - 側邊欄包含：「下一個方塊 (Next Piece)」預覽區、分數 (Score)、行數 (Lines) 顯示，以及「開始/暫停」按鈕。
   
2. **CSS 樣式**：
   - 使用深色主題 (Dark Theme)，配上霓虹感或平滑色彩的方塊。
   - 利用 CSS Grid / Flexbox 進行版面佈局，讓遊戲在各種解析度下都能置中顯示。
   - 加入微小動畫（例如消除行時的閃爍效果）。

3. **JavaScript 邏輯**：
   - **遊戲迴圈**：使用 `requestAnimationFrame` 處理方塊的定時下落。
   - **方塊資料模型**：定義 7 種經典方塊 (I, J, L, O, S, T, Z) 及其旋轉狀態矩陣，並賦予不同的鮮豔色彩。
   - **碰撞偵測**：確保方塊不會超出邊界或與已固定的方塊重疊。
   - **輸入處理**：綁定鍵盤事件，處理防抖 (Debounce) 或延遲重複 (DAS/ARR)，提升操控手感。
   - **音效引擎**：封裝一個簡易的 `SoundEngine` 類別，使用 Web Audio API 發出不同頻率和波形的聲音。

## Verification Plan

### 自動與手動測試
- **畫面載入與佈局**：在瀏覽器開啟 `index.html`，確認畫面置中、樣式現代且無破圖。
- **操作測試**：按下開始後，使用箭頭、Z、空白鍵確認移動、旋轉與硬降的邏輯正確。
- **遊戲機制測試**：
  - 方塊是否能正確堆疊？
  - 填滿一行後是否會正確消除並計分？
  - 方塊頂到天花板是否會觸發 Game Over？
- **音效測試**：各種動作是否伴隨對應的合成音效且不會延遲。
