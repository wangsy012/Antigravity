# 俄羅斯方塊 (Tetris) 開發計畫

這是一個基於純前端 (HTML/CSS/Vanilla JS) 開發的俄羅斯方塊遊戲。

## User Review Required

> [!IMPORTANT]
> **音效實作方式**：為了達到真正的「純前端」且無需依賴外部素材檔案（避免路徑問題或素材缺失），我計畫使用瀏覽器內建的 **Web Audio API** 來動態合成 8-bit 復古風格的音效（包含移動、旋轉、消除、遊戲結束等）。請問您是否同意這個作法？
> 
> **檔案結構**：我計畫將程式碼乾淨地拆分為 `index.html`, `style.css`, `script.js` 三個檔案，以便於閱讀與維護。如果您希望全部包裝在單一個 `.html` 檔案中，請告訴我。

## Proposed Changes

### 核心前端結構與樣式

#### [NEW] index.html
- 定義遊戲畫面的 DOM 結構。
- 左側：主遊戲畫布 (Canvas)。
- 右側：資訊面板，包含「下一個方塊」的預覽畫布 (Canvas)、目前分數 (Score)、行數 (Lines)、等級 (Level) 以及開始/暫停按鈕。

#### [NEW] style.css
- 採用現代化、淺色主題 (Light Theme) 的設計。
- 遊戲區域會有清新的邊框，方塊顏色採用高飽和度的色彩，以呈現精美的視覺體驗。

### 遊戲邏輯與控制

#### [NEW] script.js
- **方塊定義**：標準的 7 種 Tetromino (I, J, L, O, S, T, Z)，每種皆有獨立的鮮豔顏色。
- **矩陣與碰撞**：實作 10x20 的遊戲網格，處理方塊與邊界及已固定方塊的碰撞偵測。
- **操作控制**：
  - `←` / `→` (左/右箭頭)：左右移動。
  - `↑` (上箭頭) / `X`：順時針旋轉。
  - `Z` 或 `Ctrl`：逆時針旋轉。
  - `↓` (下箭頭)：軟下落 (加速下落)。
  - `空白鍵` (Space)：硬下落 (瞬間到底並固定)。
- **遊戲機制**：
  - 滿行偵測、消除與上方方塊下移。
  - 計分系統：依據一次消除的行數（1-4行）給予不同分數，並隨著消除行數提升等級與下落速度。
  - 遊戲結束判定 (Game Over)。
- **音效系統**：
  - 封裝 Web Audio API，產生短促的移動音、高音的旋轉音、和弦式的消除音效等。

## Verification Plan

### Manual Verification
1. 在瀏覽器中開啟 `index.html`。
2. 確認畫面渲染正確，樣式美觀。
3. 點擊「開始遊戲」後，確認方塊會自動下落。
4. 測試所有方向鍵（包含左右移動、上箭頭旋轉、下箭頭加速）。
5. 測試空白鍵（直接下落），確認方塊會瞬間著陸。
6. 測試 `Z` 鍵，確認方塊能向左（逆時針）旋轉。
7. 聆聽遊戲進行中是否有觸發相應的音效。
8. 測試方塊填滿一行後是否會正確消除並增加分數。
